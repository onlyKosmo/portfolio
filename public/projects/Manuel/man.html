Manuel d'utilisation de Leaflet.js

Pour les newbies (comme moi) ;)

1. Initialisation de la carte

Pour afficher une carte, vous devez d'abord créer un élément HTML où Leaflet pourra rendre la carte.

var map = L.map('map').setView([latitude, longitude], zoom);

2. Ajouter une couche de tuiles

Pour que la carte s'affiche avec des tuiles, vous devez ajouter une couche de tuiles comme sur OpenStreetMap :


L.tileLayer('https://{s}.tile.openstreetmap.org/{z}/{x}/{y}.png', {
    maxZoom: 19,
    attribution: '© OpenStreetMap contributors'
}).addTo(map);

3. Ajouter un marqueur

Vous pouvez ajouter des marqueurs sur votre carte et leur attacher des popups :


L.marker([latitude, longitude]).addTo(map)
    .bindPopup('Un marqueur Leaflet où je le souhaite !')
    .openPopup();

3.1 Ajouter un cercle

Vous pouvez ajouter des cercles sur votre carte et leur attacher des popups :


var circle = L.circle([latitude, longitude], {
    color: 'red',
    fillColor: '#f03',
    fillOpacity: 0.5,
    radius: 500
}).addTo(map);

3.2 Ajouter un polygone

Vous pouvez ajouter des polygones sur votre carte et leur attacher des popups :


var polygon = L.polygon([
    [latitude1, longitude1],
    [latitude2, longitude2],
    [latitude3, longitude3]
]).addTo(map);

4. Gérer les popups

Leaflet permet d'ajouter des interactions comme des clics sur la carte :


function onMapClick(e) {
    alert("Vous avez cliqué sur la carte à " + e.latlng);
}
map.on('click', onMapClick);

4.1 Attacher des popups

Leaflet permet aussi d'attacher des popups sûr des marqueurs ou des formes :


marker.bindPopup("Salut tout le monde !
Je suis un popup.").openPopup();
circle.bindPopup("Je suis un cercle.");
polygon.bindPopup("Je suis un polygon..");
